Exercícios para os Braços

Fortaleça seus Braços com os melhores exercícios. Confira abaixo!
 VOLTAR

Rosca Direta

A rosca direta é um exercício clássico para bíceps, realizado com barra. Mantendo postura ereta, flexione os cotovelos, levantando a barra até os ombros, focando na contração muscular.

Rosca Alternada

A rosca alternada é feita com halteres, alternando os braços. Com postura reta, flexione o cotovelo de um braço enquanto o outro permanece estático, focando na contração controlada do bíceps.

Rosca Martelo

A rosca martelo no banco é realizada com halteres, sentado. Segure os halteres em pegada neutra, flexione os cotovelos alternadamente, mantendo os braços alinhados, focando no braquiorradial e bíceps.

Tríceps Pulley na Corda

O tríceps pulley na corda é feito com uma corda no pulley. Segure as extremidades com as palmas voltadas para baixo, puxe a corda para baixo, estendendo os cotovelos e contraindo os tríceps.

Tríceps no Banco

O tríceps no banco é feito com as mãos apoiadas no banco e os pés no chão. Abaixe o corpo flexionando os cotovelos e empurre para cima, contraindo os tríceps.

Tríceps com Alter

O tríceps com haltere atrás da cabeça é feito com os braços estendidos acima da cabeça. Dobre os cotovelos, trazendo o haltere para trás, e estenda os braços, contraindo os tríceps.
